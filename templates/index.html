About

I am an aspiring developer (who's still learning code) with a mission to help people discover the power of code and the joy it can bring to their lives. I am basically a few steps ahead of you as we explore this learning process. I know python can seem really hard and scary but that doesn't have to be the case. The concepts just have to be broken down for us to understand them. We'll use my CARD theory to learn these concepts. We are going to break down all these concepts, like a stack of cards, master each concept and then use them all together to develop stuff, just like playing with cards. Whethere you want to develop that web app or ace your homework, I'm here to help you crush it!

The languages that you see on this app are HTML, CSS, Python and some JavaScript. By using this BootStrap template I saved myself a lot of time. Instead of creating the HTML and CSS from scratch I was able to edit the template to the way I wanted it to look like. The thing I love most about helping others learn code, is that as I help you learn, I also get better at it, and thus it benefits everyone involved.
